# 支持

> Language: [English](../../../SUPPORT.md) | zh-CN

AgentScope 是一个早期开源项目。

## 问题咨询

如果启用了 GitHub Discussions，请优先使用它。否则，请打开 question issue，并提供：

- 你的操作系统
- 你正在从哪个 AI 工具导入
- 文件夹或文件结构，不包含私有内容
- 你期望发生什么
- 实际发生了什么

## 导入问题

报告导入问题时，请避免直接分享私有 trace。请提供：

- 工具名称，例如 Codex 或 Claude Code
- 文件扩展名，例如 `.jsonl`
- 包含 3 到 10 条代表性记录的脱敏样例
- 可用时提供浏览器 console error

## 安全问题

不要为漏洞或敏感 trace 暴露打开公开 issue。请参见 [安全政策](./security.md)。
