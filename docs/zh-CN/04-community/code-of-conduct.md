# 行为准则

> Language: [English](../../../CODE_OF_CONDUCT.md) | zh-CN

## 我们的承诺

我们希望 AgentScope 成为一个对开发者、设计师、研究人员和 AI 工具构建者都友好的项目。

参与者应保持尊重、建设性和体贴。

## 期望行为

- 使用友好且包容的语言。
- 尊重不同经验水平和背景。
- 以建设性的方式给予和接受反馈。
- 将讨论聚焦在项目和待解决问题上。
- 讨论可能包含私有数据的 trace 时保持谨慎。

## 不可接受行为

- 骚扰、侮辱或人身攻击。
- 发布他人的私有信息。
- 未经许可分享私有 trace 数据。
- 恶意灌水、垃圾信息或反复偏离主题。

## 执行

当行为伤害项目或社区时，维护者可以删除评论、关闭 issue、拒绝 Pull Request 或限制参与。

## 来源说明

本文档改编自常见开源行为准则模式，并为早期项目刻意保持简短。
