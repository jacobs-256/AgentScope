# Code of Conduct

## Our pledge

We want AgentScope to be a welcoming project for developers, designers, researchers, and AI tool builders.

Participants are expected to be respectful, constructive, and considerate.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different experience levels and backgrounds.
- Give and receive feedback constructively.
- Focus discussion on the project and the problem being solved.
- Be careful when discussing traces that may contain private data.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Publishing someone else's private information.
- Sharing private trace data without permission.
- Trolling, spam, or repeated off-topic disruption.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior harms the project or community.

## Attribution

This document is adapted from common open-source code of conduct patterns and kept intentionally short for an early-stage project.
