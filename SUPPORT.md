# Support

> Language: English | [中文简体](./docs/zh-CN/04-community/support.md)

AgentScope is an early open-source project.

## Questions

Use GitHub Discussions if enabled. Otherwise, open a question issue with:

- Your operating system
- The AI tool you are importing from
- The folder or file shape, without private content
- What you expected to happen
- What happened instead

## Import problems

When reporting import problems, avoid sharing private traces directly. Instead, provide:

- The tool name, such as Codex or Claude Code
- The file extension, such as `.jsonl`
- A sanitized sample with 3-10 representative records
- Any browser console error if available

## Security issues

Do not open public issues for vulnerabilities or sensitive trace exposure. See [SECURITY.md](./SECURITY.md).
